# reamer-grocery-graphql
Go backend for groceries app

Find the font end of this project here: https://github.com/Reamer-House-Foundation/reamer-grocery-ui

# About this Project
This project is meant to help simplify tracking what groceries you have.
Eventually this app will help you know:
- Which of your favorite recipies you can make
- Is that Milk yours or your roommates?
- (while at the supermarket) Do we still have paprika in the pantry at home?
